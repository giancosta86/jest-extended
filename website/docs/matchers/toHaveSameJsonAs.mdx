# .toHaveSameJsonAs(expected)

Use `.toHaveSameJsonAs` when checking if two values have the same JSON serialization.

<TestFile name="toHaveSameJsonAs">
  {`test('passes when values have the same JSON serialization', () => {
  expect(90).toHaveSameJsonAs(90);
  expect("Dodo").toHaveSameJsonAs("Dodo);
  expect({alpha: {beta: 90}}).toHaveSameJsonAs({alpha: {beta: 90}});
});`}
</TestFile>
